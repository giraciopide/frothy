# Frothy

A clojure chat system over websocket.

## Installation

You need leiningen/jdk. Clone the repo.

## Usage

build with
	lein uberjar

then launch with
    $ java -jar undercover-0.1.0-standalone.jar [args]

### Bugs

None of course. You can't have bugs if you don't have features.
