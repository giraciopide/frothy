# Frothy

A silly sweet clojure chat system over websocket.

## Installation

Clone the repo, you'll need leiningen to build.

Build with
	lein uberjar


## Usage

build with
	lein uberjar

then launch with
    $ java -jar undercover-0.1.0-standalone.jar [args]

### Bugs

None of course. You can't have bugs if you don't have features.
